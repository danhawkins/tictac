== Welcome to TicTac

A simple JS / Coffeescript powered tic tac toe game implementation

== Getting Started

1. bundle
2. bundle exec rake db:migrate
3. bundle exec rails server
4. http://localhost:3000

== Running Test

1. bundle exec rake jasmine
2. http://localhost:8888
